# Shoulder Muscle Segmentation

## Install nnUNet package
- Remove old nnUNet
- Clone this repo
- Activate your PyTorch environment
- Run:
```
cd TDSI21-Shoulder-Muscle-Segmentation
pip install -e .
```

To use RMSProp : nnUNet_train 2d rmsPropTrainer Task500_Epaule FOLD --npz
